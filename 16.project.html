Definitionlist

ChatGPT

GPT
ChatGPT is an AI tool developed by OpenAI that helps users by answering questions, explaining concepts, writing content, and assisting with coding and problem-solving. It understands human language and responds in a conversational way, making it useful for students, teachers, and professionals. ChatGPT is widely used for learning, homework help, brainstorming ideas, and improving communication skills.
GPT is the acronym that represents how ChatGPT works. It means the model is trained in advance on large amounts of data and can generate human-like text.

Grammarly

GWA
Grammarly is an AI-based writing support tool designed to improve the quality of written communication.
It checks grammar, spelling, punctuation, sentence structure, and tone in real time. Grammarly also suggests better word choices and helps make writing more clear and professional. It is very useful for students writing assignments, teachers preparing notes, and employees sending official emails. By using AI, Grammarly helps users avoid mistakes, improve language skills, and gain confidence in writing.

Google Gemini

GAI
Google Gemini (Bard) is an AI tool developed by Google that focuses on information retrieval, content creation, and knowledge assistance.
Google Gemini (formerly Bard) is an AI tool by Google that helps users search for information, generate content, and understand complex topics.

Canva AI

CAD
Canva AI is an AI-powered design tool that allows users to create attractive visual content easily. It helps in designing presentations, posters, certificates, resumes, and social media posts without needing advanced design skills. Canva AI can automatically suggest layouts, fonts, colors, and images based on user input. This tool is widely used in education, marketing, and business because it saves time and makes professional design accessible to everyone.

GitHub Copilot is an AI coding assistant developed by GitHub in collaboration with OpenAI. It helps programmers by suggesting code, completing functions, and finding solutions while coding. Copilot supports many programming languages such as C, Python, Java, and JavaScript. Beginners use it to learn coding faster, and experienced developers use it to reduce errors and increase efficiency. By understanding the context of the code, GitHub Copilot acts like a virtual coding partner.
